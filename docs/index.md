Эта документация фиксирует проектные знания в форме, которую может регулярно читать `264.DT.Knowledge`.

## Связь с общей системой

Проект является локальным источником знаний. Общий концентратор `264.DT.Knowledge` должен забирать отсюда не все файлы подряд, а структурированные выводы: практики, продуктовые гипотезы, клиентские боли, технические решения и задачи на развитие.
