# Техническое задание: Цифровой двойник культурно-исторического развития Белорецка

## 1. Общие сведения

**Наименование проекта:** Система онлайн доступа к картированным знаниям о культурно-историческом развитии Белорецка (Белорецкого района) Республики Башкортостан

**Заказчик:** Компания "Цифровой Двойник"

**Цель проекта:** Создание культурно-социального цифрового двойника Белорецка для стандартизированной обработки неструктурированных источников данных и формирования базы знаний о факторах развития и деградации города.

**Область применения:** Культурно-историческое картирование, цифровое наследие, социально-экономический анализ территорий

## 2. Описание задач

### 2.1 Основные задачи проекта

1. **Сбор и структурирование данных**
   - Создание конвейера обработки неструктурированных источников данных
   - Формирование семантического ядра культурного развития Белорецка
   - Построение фактографической базы данных

2. **Создание знаниевого графа**
   - Моделирование связей между субъектами, объектами, событиями и территориями
   - Визуализация исторических процессов и персональных взаимосвязей
   - Интеграция мифологического контекста (эпос Урал-Батыра) с историческими фактами

3. **Разработка аналитических инструментов**
   - Интерактивные дашборды для исследования данных
   - Поисковая система по культурно-историческим объектам
   - API для интеграции с другими цифровыми двойниками

4. **Публичная платформа**
   - Веб-портал на домене detwin.ru
   - Интерфейсы для различных типов пользователей
   - Система управления контентом

## 3. Архитектура системы

### 3.1 Конвейер обработки данных

```
Источники данных → OCR/Сканирование → Категоризация → Семантический анализ → Граф знаний → Визуализация
```

**Узлы конвейера:**

1. **Узел сбора данных**
   - Сканирование архивных материалов
   - OCR обработка текстов
   - Извлечение метаданных

2. **Узел категоризации (NLP)**
   - Классификация по категориям: субъекты, объекты, территории, время
   - Извлечение именованных сущностей
   - Определение типов отношений

3. **Узел нормализации**
   - Создание справочников субъектов и объектов
   - Геокодирование территорий
   - Нормализация временных меток

4. **Узел построения графа**
   - Создание узлов и рёбер знаниевого графа
   - Расчёт весов связей
   - Валидация связей

5. **Узел аналитики**
   - Построение временных рядов
   - Анализ сетевых метрик
   - Выявление паттернов

### 3.2 Структура данных

**Основные сущности:**
- **Субъекты** - персоналии (Демидовы, Твердышевы, и др.), организации
- **Объекты** - заводы, рудники, населённые пункты, памятники
- **События** - основание предприятий, смена владельцев, исторические события
- **Территории** - административные единицы, географические объекты
- **Артефакты** - документы, фотографии, книги, статьи

**Типы отношений:**
- владелец/собственник
- управляющий/директор
- основатель
- инвестор/партнёр
- родственные связи
- географическая привязка
- временная последовательность

## 4. Технологический стек

### 4.1 Основные технологии

- **R-targets** - управление конвейером обработки данных
- **R-Shiny** - веб-интерфейс и дашборды
- **Windsurf** - среда разработки с ИИ-поддержкой
- **visNetwork** - визуализация графов
- **Plotly** - интерактивная аналитика

### 4.2 Дополнительные инструменты

- **OCR API** - распознавание текстов
- **NLP API** (GPT/Claude) - анализ и категоризация текстов
- **Neo4j/NetworkX** - хранение и анализ графов
- **PostgreSQL** - реляционная база данных
- **GitHub** - управление версиями и документацией
- **Docker** - контейнеризация приложений

## 5. Пользовательские роли

### 5.1 Роли пользователей

1. **Волонтёры-сборщики** - сбор и первичная обработка материалов

- Загрузка отсканированных материалов
- Простая категоризация документов
- Проверка результатов OCR

**Для исследователей:**
- Поиск по семантическим критериям
- Создание аналитических отчётов
- Визуализация связей и процессов

**Для краеведов:**
- Верификация данных
- Добавление экспертных комментариев
- Модерация контента

**Для публики:**
- Поиск по персоналиям, объектам, периодам
- Просмотр интерактивных карт и графов
- Изучение исторических нарративов
