## Ключевые технологии

**Обязательные:**
- `targets` - управление пайплайном
- `shiny` + `visNetwork` - интерфейс и граф
- `DT` - интерактивные таблицы
- OCR API (Tesseract) - извлечение текста
- LLM API (GPT/Claude) - категоризация

**Структура проекта:**
```
beloretsk_cultural_twin/
├── _targets.R              # Конфигурация пайплайна
├── R/                      # R функции
├── data/                   # Данные (raw, processed)
├── app/                    # Shiny приложение
└── docs/                   # Документация
```
